Roadmap

Improvements to Nucleator after its initial open source release that may be valuable. 47Lining plans to continue to contribute, together with the Community.


 


help subcommand

Implement a consistent help subcommand that provides, for each nucleator command, the manpage generated from the CLI Reference source. Similar help subcommands in git.

 

beanstalk worker tier

Worker tier Stackset

 

--ssh-config option for cage configure

Print ssh-config to stdout, rather than configuring Cage

 

include AWS Config in nucleator account setup

AWS Config provides the ability to identify non-Nucleator-provisioned resources running within Nucleator Stacksets. Providing tooling around config to make this easy for users would solve a big problem that we've heard from customers - that in complex environments it can become difficult to identify and trouble the "Dependent Resources" that preclude certain actions within your Account.
