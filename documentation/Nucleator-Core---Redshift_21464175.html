Nucleator Core - Redshift

The Redshift Stackset provides configurable Redshift Clusters within Nucleator Cages.

 

The redshift command and its subcommands are used to create and maintain Amazon Redshift clusters within Nucleator Cages.

 

This is an architecture diagram of a Redshift Stackset.
