Nucleator Update

 

This project updates the Nucleator code on the nucleator machine

Just as you need to run 'nucleator update' locally, you will need to run this project to update the nucleator machine instance.

When should you do this?

  • If code updates have been made to any of the contributed modules
  • If the site configuration has been updated
