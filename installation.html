Installation

To learn how to install Nucleator, please read the documentation.

Nucleator source code is available in the following repositories:

  • nucleator
  • nucleator-core-siteconfig: Nucleator core package that contains customer specific configuration
  • nucleator-core-redshift: Nucleator core Stackset that builds an Amazon Redshift cluster
  • nucleator-core-cage: Nucleator core Stackset that builds a Nucleator Cage including a VPC, bastion and NAT hosts
  • nucleator-core-builder: Nucleator core Stackset that creates an EC2-based application build environment with Jenkins and Artifactory instances
  • nucleator-core-beanstalk: Nucleator core Stackset that creates AWS Elastic Beanstalk instances
  • nucleator-core
