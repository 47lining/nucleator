Version 1.0

In March 2015, 47Lining released the initial version of Nucleator. Capabilities include the following commands:

  • nucleator init
  • nucleator update
  • nucleator account setup
  • nucleator account rolespec
  • nucleator cage provision
  • nucleator cage configure
  • nucleator builder provision
  • nucleator builder configure
  • nucleator beanstalk provision
  • nucleator beanstalk configure
  • nucleator beanstalk deploy
  • nucleator redshift provision
  • nucleator redshift configure

For information on usage, please read the documentation.
